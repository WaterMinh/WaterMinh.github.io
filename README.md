# Minh Ly Personal Portfolio Website

## Overview

This repository contains my personal GitHub Pages portfolio website.

The website is designed to introduce myself as a Computer Science student, present my technical skills, show my course projects, and provide useful links to my GitHub and LinkedIn profiles. The goal is to make the page useful not only for course grading, but also for teachers, classmates, and future recruiters.

Website link:

https://WaterMinh.github.io

## Purpose

The original version of this website was a simple CS351 course page. It was later redesigned into a more professional personal portfolio.

The new version focuses on:

* A cleaner and more modern visual design
* A professional summary similar to a LinkedIn profile
* Clear project cards for GitHub repositories
* Better presentation of technical skills
* Course learning progress
* GitHub and LinkedIn contact links
* Responsive design for desktop and mobile users

## Technologies Used

This website was built using:

* HTML5
* CSS3
* Bootstrap 5.3.3
* GitHub Pages
* Git and GitHub
* VS Code

Bootstrap 5.3.3 was used as the main front-end framework. It helps provide responsive layout components such as containers, navigation bars, buttons, grid layout, and cards.

Custom CSS was also added to improve the design, including:

* Hero section styling
* Card shadows
* Rounded corners

* Skill tags
* Project badges
* Timeline-style coursework section

## How to Run Locally

Because this is a static website, it does not require a backend server.

To view it locally:

1. Clone the repository.

```bash
git clone https://github.com/WaterMinh/WaterMinh.github.io.git
```

2. Open the project folder.

```bash
cd WaterMinh.github.io
```

3. Open `index.html` in a browser.

You can also open the folder in VS Code and use the Live Server extension if installed.

## Deployment

This website is deployed using GitHub Pages.

After changes are committed and pushed to the `main` branch, GitHub Pages automatically updates the website.

Common Git commands:

```bash
git status
git add index.html README.md
git commit -m "Update personal portfolio website"
git push
```
